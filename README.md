# agents

Agent files for agentic AI workflows.

## Overview

This repository defines the agent prompts and workflow instructions used for orchestrated work.

## Entry Point

Use [`agents/orchestrator.md`](agents/orchestrator.md) as the workflow entry point.

The orchestrator is responsible for coordinating the research, coding, writing, and review stages in trace format.

## Documentation First

If a project is new or its documentation is missing or unclear, documentation should be generated by default.

That includes:

* README updates
* usage notes
* docstring edits

## Files

* [`AGENTS.md`](AGENTS.md) - repo-level orchestration and trace rules
* [`agents/orchestrator.md`](agents/orchestrator.md) - orchestration workflow
* [`agents/research.md`](agents/research.md) - research stage guidance
* [`agents/coder.md`](agents/coder.md) - implementation stage guidance
* [`agents/writer.md`](agents/writer.md) - documentation stage guidance
* [`agents/reviewer.md`](agents/reviewer.md) - review stage guidance
